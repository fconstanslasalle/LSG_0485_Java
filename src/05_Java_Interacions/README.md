# 05 - Java Iteracions

## Introducció

Aquesta carpeta conté exemples de codi per aprendre a treballar amb bucles i iteracions en Java. Les iteracions són estructures de control que permeten repetir un bloc de codi múltiples vegades fins que es compleixi una determinada condició.

## Conceptes

### Tipus de Bucles en Java

#### Bucle `for`
- Estructura: `for (inicialització; condició; increment)`
- Utilitzat quan es coneix el nombre d'iteracions
- Ideal per recórrer arrays i col·leccions

#### Bucle `while`
- Estructura: `while (condició)`
- S'executa mentre la condició sigui certa
- La condició es comprova abans d'executar el bloc

#### Bucle `do-while`
- Estructura: `do { ... } while (condició)`
- S'executa almenys una vegada
- La condició es comprova després d'executar el bloc

### Conceptes Relacionats
- Iteració i control de flux
- Condicions de parada
- Variables comptador i acumulador
- Bucles niats (nested loops)
- Instruccions `break` i `continue`
