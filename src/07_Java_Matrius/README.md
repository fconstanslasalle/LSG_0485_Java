# 07 - Java Matrius

Aquesta carpeta conté exemples de codi per aprendre a treballar amb Matrius (arrays bidimensionals) en Java.

## Exemples disponibles

### 07_01 - Operacions Bàsiques amb Matrius
Exemple que demostra:
- Declaració i inicialització de matrius
- Recorregut amb bucles for aniats
- Modificació d'elements
- Suma de tots els elements
- Matriu inicialitzada amb valors literals

### 07_02 - Operacions Avançades amb Matrius
Exemple que demostra:
- Suma d'elements de cada fila
- Suma d'elements de cada columna
- Diagonal principal i secundària
- Transposició d'una matriu
