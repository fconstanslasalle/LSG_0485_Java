# 09 - Java Mètodes

## Introducció

Aquesta carpeta conté exemples de codi per aprendre a treballar amb mètodes (funcions) en Java. Els mètodes permeten organitzar el codi en blocs reutilitzables, millorant la llegibilitat i mantenibilitat del programa.

## Conceptes

### Mètodes en Java
- Declaració de mètodes
- Paràmetres i tipus de retorn
- Mètodes amb i sense valor de retorn (void)
- Crida de mètodes
- Àmbit de variables (scope)
- Mètodes estàtics vs mètodes d'instància
- Sobrecàrrega de mètodes (overloading)

### Avantatges dels Mètodes
- Reutilització de codi
- Facilita la lectura i comprensió
- Simplifica el manteniment
- Permet descomposar problemes complexos
