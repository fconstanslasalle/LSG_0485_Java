# 08 - Java Strings i ArrayList

## Introducció

Aquesta carpeta conté exemples de codi per aprendre a treballar amb Strings i ArrayLists en Java. Els Strings són cadenes de text immutables, mentre que els ArrayLists són col·leccions dinàmiques que permeten emmagatzemar elements de forma flexible.

## Conceptes

### Strings
- Declaració i inicialització de Strings
- Concatenació de cadenes de text
- Mètodes principals: `length()`, `charAt()`, `substring()`, `indexOf()`, `toLowerCase()`, `toUpperCase()`
- Comparació de Strings amb `equals()` i `compareTo()`
- Immutabilitat dels Strings

### ArrayList
- Declaració i inicialització d'ArrayList
- Operacions bàsiques: `add()`, `get()`, `set()`, `remove()`, `size()`
- Recorregut d'ArrayList amb bucles
- Cerca d'elements
- ArrayList vs Arrays tradicionals
