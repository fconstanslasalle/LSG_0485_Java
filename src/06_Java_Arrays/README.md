# 06 - Java Arrays

Aquesta carpeta conté exemples de codi per aprendre a treballar amb Arrays en Java.

## Exemples disponibles

### 06_01 - Operacions Bàsiques amb Arrays
Exemple que demostra:
- Declaració i inicialització d'arrays
- Recorregut amb bucle for clàssic
- Modificació d'elements
- Suma d'elements d'un array
- Array de Strings

### 06_02 - Cerca i Operacions amb Arrays
Exemple que demostra:
- Cerca lineal d'un element
- Trobar el valor màxim
- Trobar el valor mínim
- Calcular la mitjana
- Comptar ocurrències d'un valor
